# Experiment Results

This folder collects experiment records and result summaries used for the paper draft.

- `experiment_summary.md`: reproducibility record with commands, parameters, cache paths, logs, and tables.
- `experiment_conclusion.md`: paper-facing summary of main results, ablation conclusions, writing guidance, and risks.

Raw logs remain in the project-level result folders such as `cached_results/`, `ablation_results/`, `sweep_results/`, and `results/`.
